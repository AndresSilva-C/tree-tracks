TreeTracks is a personal project I undertook from scratch to help me maintain my coding skills while also learning new ones!

Please take a look at: https://treetracks.xyz/

Features include:
- access to browsing and searching for everyone, even without an account
- creating an account in order to save and view your favorite trees
- viewing individual tree species details

Tech stack used:
- Node.js
- Express
- PostgreSQL
- React wrapped with Vite, and routing with react-router
- Tailwind CSS
- Python scripting and ethical web scraping for obtaining some of the data
- My own RESTful API
- External API integration: GBIF API for occurrence mapping
